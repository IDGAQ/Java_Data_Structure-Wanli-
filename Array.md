# Array
---

## Declare:
```java
String[] cars;
int[] myNum={10,20,30,40};
```

## Acess:
```java
System.out.println(cars[0]);
```

## Change Element:
```Java
cars[0]="Opel";
```

## Length:
```Java
cars.length;
```

## Loop:
```Java
for(int i=0;i<cars.length;i++)
{
  System.out.println(cars[i]);
}

for(String a: cars)
{
  System.out.print(a);
}
```
