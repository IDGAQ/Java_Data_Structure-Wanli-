# Multi-dimensional Array
---
## Declare:
```java
int [][] myNumbers;
```

## Access:
```java
println(cars[0][0];
```

## Change Element:
```java
myNumbers[0][0]=1;
```

## Length:
```java
myNumbers.length;
myNumbers[0].length;
```

## Loop:
```java
for(int i=0;i<myNumbers.length;i++)
{
  for(int j=0;j<myNumbers[0].length;j++)
  {
    System.out.println(myNumbers[i][j];
  }
}

```
